## 🚀 How to Push to GitHub

### Step 1: Initialize Git (if not already done)

```bash
cd /home/sanket/CascadeProjects/windsurf-project-6/SWTMeet
git init
```

### Step 2: Add Remote Repository

```bash
# Replace with your GitHub repository URL
git remote add origin https://github.com/YOUR_USERNAME/SWTMeet.git
```

### Step 3: Add All Files

```bash
git add .
```

### Step 4: Commit

```bash
git commit -m "Initial commit: SWTMeet video conferencing platform

- Complete video meeting platform with WebRTC
- Supabase authentication and database
- Real-time chat and participant tracking
- Waiting room and host controls
- Screen sharing support
- Smart adaptive layout
- Responsive design
- Guest access support"
```

### Step 5: Push to GitHub

```bash
# For first push
git branch -M main
git push -u origin main

# For subsequent pushes
git push
```
